#README

Pet project for minio

Start minio:
1) Docker: \
 `docker-compose -f docker-compose.yaml up -d`
2) CMD: \
 `C:\Java\minio\minio.exe server C:\Java\minio\Data --console-address ":9001"`
   
Start Minio console: \
http://127.0.0.1:9001/ \
login: admin \
psw: password \


   
Start app: \
http://127.0.0.1:8082/swagger-ui.html
